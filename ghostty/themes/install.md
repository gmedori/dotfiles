### [Ghostty](https://ghostty.org)

#### Installation

1. Create the configuration directory if it does not exist:

```sh
mkdir -p ~/.config/ghostty
```

2. Create or edit the configuration file:

```sh
nano ~/.config/ghostty/config
```

3. Choose your preferred variant and copy/paste the corresponding content.

#### Activating theme

1. Save and exit your editor (`Ctrl+X`, then `Y`, then `Enter` for nano).
2. Restart Ghostty for changes to take effect.
